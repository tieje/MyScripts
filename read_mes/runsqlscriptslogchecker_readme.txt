requirements
- python 3.7
- must be in the same folder as runsqlscript.py
- if the sql script spools to a ".txt" file and not a ".log" file, this script will not work as intended. In this case, rename the ".txt" file to ".log" file and run the mhklogchecker.py on that directory to check for errors
